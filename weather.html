SkyCaster – Web Weather App

View live site ↗

APL project · OpenWeather API · UI/UX focus · Iterative development

Created a weather application using a GenAI-based workflow during my internship (APL).

Worked with API data to display weather conditions and forecasts, with a focus on clarity and usability. Tested the functionality and refined the interface from a user perspective.

Further developed and deployed the application, refining the visual design and improving the interface step by step.

What I focused on

  • clear and minimal user interface
  • user feedback and input validation
  • using API data to display real-time weather information
  • iterating on UI/UX using AI-generated code

Key features

  • search by city name
  • real-time weather data from OpenWeather API
  • 5-day forecast
  • visual feedback and error handling
  • Background subtly adapts to current weather conditions

What I learned

This project gave me hands-on experience working with APIs, including retrieving and using an API key to display real-time data.

It also helped me explore how AI can be used as a tool in development, not to replace understanding, but to support iteration, testing ideas, and improving the user experience.
